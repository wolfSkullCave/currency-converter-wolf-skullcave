## Installation

```console
pip install currency-converter-wolf-skullcave
```

## Usage

To view the exchange rate (USD to ZAR)
```console
currency
```

To convert an amount from USD to ZAR
```console
currency <amount>
```

Convert from EUR to ZAR
```console
currency --currency eur <amount>
```
OR
```console
currency -e eur <amount>
```
